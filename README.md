# SysDraw Web App 🛠️

A full-stack web application designed to act as a Computer-Aided Software Engineering (CASE) tool. This project facilitates software design, architecture planning, and system diagramming directly within the browser.

## 🚀 Key Features
* **Interactive Modeling:** Create, edit, and manage software architecture diagrams.
* **Component Management:** Track system entities, relationships, and data flows.
* **Persistent Storage:** Saves project states and user data using a custom JSON-based storage system.

## 💻 Tech Stack
* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** File-based JSON storage
